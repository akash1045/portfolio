# Akash Chavan — Portfolio

A sleek, dark-themed developer portfolio built with React. Fully responsive across mobile, tablet, and desktop.

## 🚀 Quick Start

```bash
cd akash-portfolio
npm install
npm start
```

Opens at `http://localhost:3000`

## 📦 Build for Production

```bash
npm run build
```

Outputs to the `build/` folder — deploy to Vercel, Netlify, or GitHub Pages.

## ✏️ Customization

All your personal data lives in **`src/data/resume.js`** — edit that file to update any content.

## 🎨 Design

- Dark theme with cyan accent (`#00e5ff`) and purple (`#7c3aed`)
- Fonts: **Syne** (headings) + **DM Sans** (body)
- Fully responsive — mobile-first grid layouts
- Smooth animations, hover effects, grain texture overlay
